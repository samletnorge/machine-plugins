# rag_support

Reusable retrieval and generation support for Machine.

## Provides

- categories for `chunker`, `reranker`, `metadata_extractor`, and `rag_pipeline`
- always-local chunkers plus optional provider-backed retrieval helpers
- `RAGPipeline` and `GraphRAG`
- adapters and shared retrieval models

## Key Files

- `manifest.json`
- `__init__.py`
- `pipeline.py`
- `graph.py`
- `adapters.py`
- `models.py`

## Role

This plugin provides reusable retrieval components that can be composed into domain agents and workflows.
