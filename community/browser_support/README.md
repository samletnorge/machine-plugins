# browser_support

Browser automation support for Machine runtimes.

## Provides

- the `browser` category
- a common browser abstraction and validated browser result types
- a Playwright-backed browser implementation
- a Stagehand-style wrapper for natural-language selector resolution
- helper wrappers that expose browser actions as agent-friendly tools

## Key Files

- `manifest.json`
- `__init__.py`
- `base.py`
- `playwright_browser.py`
- `stagehand_browser.py`
- `tools.py`

## Role

This plugin provides reusable web-automation building blocks for agents and interactive runtime tooling.
