# provider_google_gemini

Google Gemini model provider for Machine.

## Provides

- the `model_provider/google-gemini` implementation
- pydantic-ai backed Gemini integration adapted to Machine response models

## Key Files

- `manifest.json`
- `__init__.py`
- `provider.py`

## Role

This plugin registers Google Gemini as a shared `model_provider` backend.
