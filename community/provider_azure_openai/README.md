# provider_azure_openai

Azure OpenAI model provider for Machine.

## Provides

- the `model_provider/azure-openai` implementation
- pydantic-ai backed Azure OpenAI integration with API-key or token-auth support
- Machine-compatible normalized model responses

## Key Files

- `manifest.json`
- `__init__.py`
- `provider.py`

## Role

This plugin registers Azure OpenAI as a shared `model_provider` implementation.
