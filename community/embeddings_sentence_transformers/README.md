# embeddings_sentence_transformers

Local sentence-transformers embedding provider for Machine.

## Provides

- the `embedding/sentence_transformers` implementation
- lazy loading of a configured sentence-transformers model
- configurable local embedding generation and batch sizing

## Key Files

- `manifest.json`
- `__init__.py`

## Role

This plugin provides a fully local embedding option for projects that want local vector generation without a hosted API.
