Install

Get Machine

Bootstrap the Machine CLI the usual way, then move between the TUI, per-project runtime server, and Studio control plane.

curl -fsSL https://gist.githubusercontent.com/valiantlynx/c3eaf552adf9aecff7c0366a25ff1e99/raw/install.sh | bash
Inside a project

Main commands

machine

Launches the TUI for the current project perspective.

machine dev

Starts the runtime server for the current project.

machine studio

Starts Studio for the current project context and mounts it at /_studio.

Outside a project

Bootstrap flow

machine

Still opens the TUI, but project-aware surfaces stay offline without a detected Machine project.

machine init

Scaffolds a new Machine project with a default plugin list and Studio config.
